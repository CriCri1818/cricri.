Android 3.0 Honeycomb

⬅ Volver

Información general

Android 3.0, conocido como Honeycomb, fue una versión exclusiva para tablets. Google decidió crear esta versión para aprovechar pantallas grandes, separándola temporalmente de los teléfonos. Introdujo una interfaz completamente nueva con botones virtuales en pantalla, barra de sistema inferior y multitarea visual. Honeycomb mejoró la aceleración por hardware y el uso de procesadores multinúcleo. Aunque no fue muy popular y tuvo una vida corta, fue fundamental para experimentar con diseños y funciones que más adelante se integrarían en otras versiones de Android.

Características principales

  • Interfaz optimizada para pantallas grandes
  • Nueva barra de sistema
  • Mejor multitarea
